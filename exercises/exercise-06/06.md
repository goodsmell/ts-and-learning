## 키워드 단순 학습

함수 오버로딩이란?
특정 입력 조합에 따른 특정 출력 결과를 미리 여러개 선언해 두는 것.

제너릭 함수
어떤 함수의 매게변수나 반환 값에 다양한 타입을 넣고 싶을 때 제네릭을 사용할 수 있다.

```
function identity<T>(arg: T): T {
    return arg;
}

const output1 = identity<string>("hello"); // T는 string
const output2 = identity(123); // TypeScript가 T를 number로 자동 추론.
```

`<T>` : T라는 타입의 변수를 쓸거나는 뜻
`arg:T`: 입력받는 인자의 타입이 T라는 뜻.
`: T` : 이 함수가 뱉는 결과값도 T타입이라는 뜻

## 문제에 적용시켜 해결한 방법

문제점 : 이전까지는 personType에 user을 넣든, admin을 넣든 무조건 Person[]을 반환했다. 그럼 사용할 때마다 이게 유저인가 어드민인가 또 체크해야하는 상황이 벌어진다.

해결책 : filterPersons을 오버로딩 하여, personType이 user일 땐 User[]을, admin일 때는 Admin[]을 반환하도록 하였습니다.
